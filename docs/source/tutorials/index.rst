General Setup
-------------

Before starting any tutorial, make sure you have:

1. Python 3.9 or higher installed
2. Cloned the repository:

   .. code-block:: bash

      git clone https://github.com/saint2706/Games.git
      cd Games

3. Installed dependencies:

   .. code-block:: bash

      pip install -r requirements.txt

Optional Dependencies
~~~~~~~~~~~~~~~~~~~~~

For GUI support, you may need:

* **tkinter**: Usually included with Python, but may need separate installation on Linux
* **pygame**: For sound effects in Uno (``pip install pygame``)
